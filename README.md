# FreelanceHub

**FreelanceHub** es una herramienta de consola (CLI) en **Python** diseñada para que desarrolladores independientes gestionen sus proyectos, tiempos y finanzas en un solo lugar.

## 🛠️ Funciones Principales
* **Gestión de Tareas:** Creación de carpetas de proyecto y listas de tareas (Pendiente/En Proceso/Finalizado).
* **Control de Tiempo:** Registro de horas trabajadas y generación de reportes.
* **Módulo Financiero:** Calculadora de presupuestos y conversor de moneda (USD/ARS).
